*****************************************
*	Requisitos Previos		*
*****************************************

Las dependencias previas que debe contener el sistema son:

	-pthread (ultima version)
	-ogre versión 1.9
	-mygui versión 3.2.3
	-bullet versión 2.83
	-ogrebullet (ultima version)
	-mxml versión 1.5
	-ois versión 1.3.0
	-sdl versión 1.2
	-boost_system versión 1.58
	-OpenGL versión 1.2
	-wiimote-ng version 0.5

*****************************************
*	Sistemas Operativos testeados	*
*****************************************
	Linux Ubuntu 32 Bits
	Linux RedHat 64 Bits	

*****************************************
*	Repositorio 			*
*****************************************
El repositorio del proyecto es el siguiente:

El repositorio contiene tanto el código fuente como los siguientes elementos adicionales:
	Proyectos de diseño gráfico GIMP:

	Documentación:
	https://github.com/jalcolea/cars/tree/master/DOCUMENTATION

*****************************************
*	Compilación 			*
*****************************************

1	Descargar e instalar las dependencias indicadas en el apartado "Requisitos previos"
2	Descargar el repositorio github del proyecto
3	Entrar en el directorio principal del proyecto "cars"
4	Ejecutar make 
5	Una vez finalizada la compilación ejecutar el binario creado "cars" ubicado en el directorio principal del proyecto.
